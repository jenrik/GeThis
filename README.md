# GeThis
GeThis (procounced "get this") is a web controllet utility to download files to a remote server.

## Running
First install Node.js and npm from your package manager of choice. Then you can open the repositorie in a terminal and type "npm install", then "node index.js" to start GeThis.

Note: On some system you will need to start with nodejs instead of node.
